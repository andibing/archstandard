Tous les téléchargements en un seul endroit. Les modèles peuvent être importés dans un LLM pour une rédaction assistée par IA, ou remplis directement.

## Standard complet

Le standard complet Architecture Description Standard en un seul document :

| | Format | Téléchargement |
|:--:|--------|----------------|
| | Word (.docx) | **[ads-standard-v1.docx](/v1/ads-standard-v1.docx)** |
| | Open Document (.odt) | **[ads-standard-v1.odt](/v1/ads-standard-v1.odt)** |

## Modèles SAD vierges

Commencez un nouveau Document d'Architecture de Solution :

| | Format | Téléchargement | Idéal pour |
|:--:|--------|----------------|-----------|
| | Word (.docx) | **[sad-template.docx](/v1/templates/sad-template.docx)** | Remplissage direct |
| | Markdown (.md) | **[sad-template.md](/v1/templates/sad-template.md)** | Git, wikis, Confluence |
| | YAML (.yaml) | **[sad-template.yaml](/v1/templates/sad-template.yaml)** | CI/CD, outillage, LLMs |
| | JSON (.json) | **[sad-template.json](/v1/templates/sad-template.json)** | Validation de schéma, LLMs |
