## À propos de cet exemple

Une plateforme d'API REST cloud-native orientée client sur AWS. Cet exemple illustre un SAD au niveau **Comprehensive** avec des attributs de qualité complets, une vue sécurité détaillée (modèle de menace inclus), des intégrations étendues et une traçabilité de conformité.

Idéal pour comprendre comment documenter un système critique de production avec exigences réglementaires.

| Caractéristique | Valeur |
|-----------------|--------|
| **Profondeur de documentation** | Comprehensive |
| **Criticité** | Tier 1 (Critique) |
| **Hébergement** | AWS EKS |
| **Score global de conformité** | 5 / 5 |
