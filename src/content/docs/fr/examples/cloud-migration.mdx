## À propos de cet exemple

Une application legacy on-premises qui migre vers Azure. Cet exemple illustre les sections de migration de l'ADS — les 6 R (Rehost, Replatform, Refactor, Repurchase, Retire, Retain), l'architecture de l'état actuel et le plan de transition.

Idéal pour comprendre comment documenter un projet de modernisation ou de migration cloud, avec analyse de l'existant et stratégie de bascule.

| Caractéristique | Valeur |
|-----------------|--------|
| **Profondeur de documentation** | Recommended |
| **Criticité** | Tier 2 (Élevée) |
| **Hébergement cible** | Azure PaaS |
| **Score global de conformité** | 3 / 5 |
