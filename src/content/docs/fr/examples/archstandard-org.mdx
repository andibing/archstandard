## À propos de cet exemple

Il s'agit d'un **méta-exemple** : l'Architecture Description Standard documentant son propre site web. Il démontre que même les projets les plus simples — un site de documentation statique sans backend, sans base de données et sans comptes utilisateurs — peuvent utiliser l'ADS au niveau **Minimum** sans surcoût inutile.

Lorsqu'une section de profondeur Minimum n'est réellement pas applicable (par ex., aucune connectivité interne entre composants n'existe), cela est noté brièvement et l'on passe à la suite.

| Caractéristique | Valeur |
|-----------------|--------|
| **Profondeur de documentation** | Minimum |
| **Criticité** | Tier 5 (Minimale) |
| **Hébergement** | Cloudflare Pages |
| **Score global de conformité** | 3 / 5 |
