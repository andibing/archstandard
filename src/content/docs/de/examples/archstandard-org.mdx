## Über dieses Beispiel

Dies ist ein **Meta-Beispiel**: Der Architecture Description Standard dokumentiert seine eigene Website. Es zeigt, dass selbst die einfachsten Projekte — eine statische Dokumentationsseite ohne Backend, Datenbank und Benutzerkonten — den ADS auf der Tiefe **Minimum** verwenden können, ohne unnötigen Overhead.

Wo ein Abschnitt der Tiefe Minimum tatsächlich nicht zutrifft (z. B. keine interne Komponentenkonnektivität existiert), wird das kurz vermerkt und zum nächsten Punkt übergegangen.

| Merkmal | Wert |
|---------|------|
| **Dokumentationstiefe** | Minimum |
| **Kritikalität** | Tier 5 (Minimal) |
| **Hosting** | Cloudflare Pages |
| **Gesamt-Konformitätsbewertung** | 3 / 5 |
