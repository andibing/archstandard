## Über dieses Beispiel

Dies ist ein **fiktives, aber realistisches** Solution Architecture Document für ein internes Mitarbeiterverzeichnis bei Meridian Financial Services. Es demonstriert den ADS-Standard auf der Dokumentationstiefe **Recommended**, geeignet für eine geradlinige interne Tier-4-Anwendung auf dem Weg in die Produktion.

Nutzen Sie es als Referenz beim Erstellen Ihres eigenen SAD. Struktur, Feldnamen und Enum-Werte folgen exakt dem Standard.

| Merkmal | Wert |
|---------|------|
| **Dokumentationstiefe** | Recommended |
| **Kritikalität** | Tier 4 (Niedrig) |
| **Hosting** | Azure PaaS |
| **Gesamt-Konformitätsbewertung** | 3 / 5 |
